# Node.js Web Server

A simple web server built with Node.js.

## Features

- Serves static files
- Handles basic routing
- Easy to configure and extend

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd express_web_server
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the server:**

   ```bash
   node server.js
   ```

4. **Visit in your browser:**
   ```
   http://localhost:3500
   ```
